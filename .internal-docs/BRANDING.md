# Branding

## Criterio visual

Xzp debe verse tecnico, sobrio y legible. La identidad no necesita parecer una suite comercial; necesita comunicar control y claridad en terminal.

## Principios

- contraste suficiente
- color como apoyo, no como ruido
- titulos cortos
- bloques compactos y repetibles

## Restricciones

- evitar banners excesivos en flujos frecuentes
- no depender del color para entender estados
- no mezclar una identidad demasiado ornamental con comandos de auditoria
