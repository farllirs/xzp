# Release: Xzp v1.2.0

## 📊 Resumen Operativo

La versión `1.2.0` es una actualización mayor que introduce capacidades críticas de conectividad entre entornos y una renovación completa de la experiencia visual en procesos de larga duración. Esta versión transforma a Xzp de una utilidad de inspección a un puente de datos integral para Termux y Linux.

---

## 🚀 Hitos de la Versión

### Xzp Bridge (Conectividad Cruzada)
- Implementación de destinos especiales `'linux'` y `'termux'` para el comando de pegado (`-p`).
- Mapeo automático de sistemas de archivos de `proot-distro` desde el host de Termux.
- Sincronización del portapapeles a través de un puente persistente en el almacenamiento compartido de Android.

### Smart Install & Progress Bar
- Nuevo motor de instalación silenciosa que captura y procesa la salida de gestores de paquetes (npm, pip, composer, etc.).
- Interfaz visual unificada con barra de progreso `#` y actualización dinámica de paquetes.
- Resumen post-instalación con desglose de éxitos y fallos.

### Sistema de Actualización Global
- Comprobación automática de versiones contra el registro de NPM.
- Flujo interactivo de actualización con opciones de Update, Skip y Quit.
- Proceso de actualización global integrado con la nueva barra de progreso visual.

### Seguridad y Blindaje
- Saneamiento de salidas de terminal para prevenir inyecciones ANSI.
- Sistema de redacción automática de secretos para ocultar Tokens y API Keys en logs.
- Validación estricta de permisos y rutas en operaciones de puente.

---

## 🛠️ Mejoras Técnicas

- **UI Modular:** Creación de `src/ui/progress.js` como utilidad compartida para procesos asíncronos.
- **Arquitectura de Puente:** Implementación de `getLinuxDistroRoots` para el escaneo dinámico de distribuciones.
- **Refactorización de Documentación:** Renovación total de documentos internos y README bajo el criterio de estilo "Soberbio y Técnico".

---

## 📝 Nota Técnica de Release

La `1.2.0` representa la consolidación de la visión de Xzp como una capa de control profesional. La capacidad de operar archivos entre Android y Linux sin salir de la shell, junto con la nueva transparencia visual de las instalaciones, sitúa a esta versión como el estándar de estabilidad y potencia para el ecosistema Nyxur.
