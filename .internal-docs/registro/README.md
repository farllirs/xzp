# Registro Histórico de Versiones

Repositorio centralizado de notas técnicas y cambios significativos por versión de Xzp.

---

## 📖 Estructura del Registro

- **Foco Técnico:** Cada entrada debe resumir cambios con impacto real en la operación del sistema.
- **Contexto Claro:** Priorizar la claridad del estado del proyecto sobre changelogs extensos.
- **Categorización:** Organización de cambios en Novedades, Mejoras y Parches según sea necesario.
